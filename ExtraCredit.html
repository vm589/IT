Do's and Don't when building a website

There are many ways to build a website but for it to grab people' attention is another thing. Here are 5 general do's and don'ts when you start up building your website

Things you should do

  • Always use "div" as its helpful as even though it does not have any visual effect on the website it helps organizing the code
  • The use of break or br is helpful especially since you cannot really know how a website will look when typing the paragaph out.
    It will come down too the format of the website so using it helps keep it more in order to restart it on a new line
  • Using headers is very important as it helps give a title or introduction to the page.
  • Always remember to use "a" tags for whenever you want to provide an external link to another site for whatever reason
  • Linking images are more complicated so its important to memorize the steps.
    The source file or src is placed at the start and is where you put the image link, you then put alternative text and although it is optional you could put the width and height and adjust it.

to avoid

  • Not naming your headings in the right order. For example going from h1 to h4
  • Confusing the title with a heading is another common mistake
  • Confusing li with ol in terms of listing
  • Trying to mix the css with the html. First finish the html coding to leave it less jarring to deal with
  • It is better to use an external css folder then incline
  • Overall just not having a clear image in mind, always have a clear goal so the website does not come out looking terrible.
